# Hi, I'm Suyog 👋 — Portfolio

A personal portfolio website built with vanilla **HTML, CSS &amp; JavaScript** — no frameworks, no build step. Features an animated "aurora glassmorphism" theme with floating gradient blobs, glowing glass cards, animated skill bars, and tilt-on-hover project cards.

**🔗 Live site:** [suyog-sonawane.github.io](https://suyog-sonawane.github.io/)

---



## ✨ Features

- Animated hero with typing effect and floating tech chips
- Scroll-triggered reveal animations
- Animated, self-rated skill progress bars
- Tilt-on-hover project cards (mouse-tracking 3D effect)
- Fully responsive — down to mobile
- Respects `prefers-reduced-motion` for accessibility
- Zero dependencies — pure HTML/CSS/JS, loads fast

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, animations, glassmorphism)
- Vanilla JavaScript (IntersectionObserver, no libraries)
- [Google Fonts](https://fonts.google.com/) — Sora, Inter, JetBrains Mono

## 📁 Project Structure

```
├── index.html          # Main page markup
├── style.css            # All styling (theme, layout, animations)
├── script.js             # Typing effect, scroll reveal, tilt cards, counters
└── assets/
    └── profile.jpg      # Profile photo
```

## 🚀 Getting Started (run locally)

1. Clone the repo
   ```bash
   git clone https://github.com/Suyog-Sonawane/Suyog-Sonawane.github.io.git
   ```
2. Open the folder in **VS Code**
3. Install the **Live Server** extension (by Ritwick Dey)
4. Right-click `index.html` → **Open with Live Server**

No build tools, no `npm install` — it just runs.

## 🌐 Deployment

This site is deployed with **GitHub Pages**, served directly from the `main` branch. Any push to `main` updates the live site automatically within a minute or two.
